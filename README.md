Question-01: What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Ans: getElementById is used to select a single element by its id from HTML.
getElementsByClassName is used to find all element that shear with same className form html.
querySelector is used to find first element that matches from CSS selector and
querySelectorAll is used to find all element that matches from Css selector.
